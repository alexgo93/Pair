# Installation

composer require alexgo93/php_pairs

# Description

This is the library for pair data structure.

# Usage

#### Create pair:

> $pair = new Pair($value1, $value2);

#### Get first value:

> $pair->car();

#### Get second value:

> $pair->cdr();
